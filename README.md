# 🌤️ Weather App

A simple desktop weather application built with Python and PyQt5.
Enter any city name to get real-time weather data powered by the OpenWeatherMap API.

## Features
- Search current weather by city name
- Displays temperature in Celsius
- Shows a weather emoji based on conditions (thunderstorm, rain, snow, fog, etc.)
- Displays a short weather description
- Handles common errors (invalid city, no connection, timeouts, etc.)

## Tech Stack
- Python
- PyQt5 (GUI)
- OpenWeatherMap API

## Setup
1. Clone the repo
2. Install dependencies:
   pip install requests PyQt5
3. Run the app:
   python weather_app.py
